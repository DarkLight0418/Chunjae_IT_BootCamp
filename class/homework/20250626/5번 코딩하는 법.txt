5번 코딩 과제 하는 법

1. 코드 맨 윗 줄 파일 클래스 이름이 A ex) class A
=> 5번 쳐야 하니 클래스 이름 뒤에 각각 1, 2, 3, 4, 5 붙이기!
	ex) 4번째의 경우 'class A4'

2. 메인 메소드(public static void main(String[] args))가 있어야 프로그램을 실행시킬 수 있음.
메인 메소드에 ex) ' ' 형태로 코드 넣어주기

=> 메인 메소드 중괄호 {} 안에서 객체(클래스를 통해 만들어진 구현체(인스턴스)) 생성
=> ex) 3번째의 경우 'A3 a = new A3();


3. 다 작성 후 cmd(명령 프롬프트)로 컴파일 후 실행 => .class 파일들 만들어짐
-> 컴파일 후 오류 없으면 실행시켜서 잘 출력되는지 확인

4. 제출할 때는 폴더에 .java 파일들만 집어넣고(.class 파일들은 삭제해도 괜찮음) 압축
(폴더 위에 오른쪽 마우스 클릭 - 압축 대상... - Zip 파일)
, 이름은 본인이름_날짜 ( ex)김누구_0626 )

5. 메콩이 클래스룸(FULLSTACK12기_클래스룸)이나 김형수 선생님 개인 메시지로 파일 첨부하여 제출(글쓰기(채팅 보내는 거) 란의 클립 표시 클릭해서 첨부하거나, zip파일 선택하고 메콩이로 파일 끌어서 올린 후 보내기)


'' 예시 코드 ''
class A1 {
    void m1() {  // 메소드
        System.out.println("m1");
    }

    public static void main(String[] args) {
        // 메인 메소드(항상 여기다가 객체 생성해서 넣어주기)
        A1 a = new A1();  // 객체 생성하는 방법
        a.m1();  // 객체에 있는 메소드 사용하는 방법
    }
}
